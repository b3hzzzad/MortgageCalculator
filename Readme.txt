# Mortgage Calculator

A simple Java console application to calculate monthly mortgage payments and track remaining balance over time.

## Features
- Input loan amount, annual interest rate, and period (years)
- Calculates monthly payment using standard mortgage formula
- Prints remaining balance after each month
